# Многобрендовая платформа Skilla

## Структура проекта

Это единая кодовая база, которая позволяет быстро создавать веб-сайты для разных брендов. Все различия управляются через CSS переменные и данные.

### Основные компоненты

**Компоненты секций** (`src/components/sections/`):
- `SiteHeader.astro` - Заголовок сайта с навигацией
- `Hero.astro` - Баннер/герой секция
- `Services.astro` - Список услуг
- `CalculatorForm.astro` - Форма калькулятора стоимости
- `BlogPreview.astro` - Превью последних блог-постов
- `KP.astro` - Секция для коммерческого предложения
- `News.astro` - Новости компании
- `FAQ.astro` - Часто задаваемые вопросы
- `PriceTable.astro` - Таблица цен
- `Advantages.astro` - Преимущества компании
- `Testimonials.astro` - Отзывы клиентов
- `HowWeWork.astro` - Процесс работы
- `Brands.astro` - Партнеры/сотрудничающие бренды
- `Thanks.astro` - Благодарности
- `Map.astro` - Карта расположения
- `NewsList.astro` - Список статей/новостей
- `BlogPost.astro` - Отдельная статья
- `Contacts.astro` - Контактная информация
- `SiteFooter.astro` - Подвал сайта
